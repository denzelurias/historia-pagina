# Referencias de diseño

Guarda aquí imágenes, capturas, PDFs o notas que quieras usar como inspiración visual para el proyecto.

## Estructura sugerida

- `layout/`: composición general, secciones, hero, navegación.
- `color/`: paletas, atmósferas, texturas, materiales.
- `typography/`: estilos de títulos, jerarquías, combinaciones tipográficas.
- `interactions/`: timelines, tarjetas expandibles, menús, detalles interactivos.

## Cómo nombrar archivos

Usa nombres descriptivos para que luego sea fácil referirnos a ellos.

Ejemplos:

- `layout/museo-editorial-01.png`
- `color/pergamino-oscuro-02.jpg`
- `typography/titulos-serifa-01.png`
- `interactions/timeline-vertical-01.png`

## Notas útiles

Si quieres, también puedes crear un archivo `.md` junto a una imagen para explicar qué te gustó de esa referencia.

Ejemplo:

- `layout/museo-editorial-01.png`
- `layout/museo-editorial-01.md`

Dentro de esa nota puedes poner cosas como:

- qué te gusta;
- qué no quieres copiar tal cual;
- qué parte sí quieres adaptar al proyecto.
